## [2.3.79] - 2026-03-18

### Added
- Initial release of Dart port
- Transliteration between Indic scripts (Devanagari, Bengali, Gujarati, Gurmukhi, Kannada, Malayalam, Oriya, Tamil, Telugu)
- Support for Romanization schemes (IAST, Harvard-Kyoto, ITRANS, SLP1, Velthuis, WX, etc.)
- Automatic script detection
- Toggle and suspend functionality for preserving text during transliteration
- Works on all Dart platforms (VM, Flutter, Web)
